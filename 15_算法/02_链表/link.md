#### 链表：是由一组不必相连的内存结构，按照特定的顺序连接再一起的抽象数据类型

#### 分类： 单项链表、双向链表、环状链表

#### 核心操作：插入、删除、查找遍历

#### 单链表 --> 只能通过 next 往下走

- 第一个节点是链头
- 最后一个节点是链尾

```
// 节点：
{
    value，
    next
}
```

#### 双向链表 --> 能够通过 next 往下走，也能通过 prev 往上走

```
{
    value,
    prev,
    next
}
```

#### 单向循环链表 --> 通过 next 构成了一个环

#### 双向循环链表 --> 通过 next 形成了一个环， prev 也构成了一个圆环
